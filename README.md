# Innovative Approaches For Invertebrate Biodiversity Monitoring

This repository contains a complete pipeline for the analysis of metabarcoding data targeting soil invertebrates from environmental DNA (eDNA) samples. The material was developed for the course *“Innovative Approaches for Invertebrate Biodiversity Monitoring”* (Prof. Matteo Montagna, University of Naples “Federico II”).

## Overview

The pipeline provides a reproducible workflow to process raw sequencing data and perform basic ecological analyses. It is organized into three parts:

### 📦 [Part 1](https://github.com/MontagnaLab/Innovative-Approaches-4-Invertebrate-Biodiversity-Monitoring/blob/main/notebooks/notebook1.md) — Setup and Data Retrieval

* Installation of required software
* Configuration of the computational environment
* Download of example dataset

### ⚙️ [Part 2](https://github.com/MontagnaLab/Innovative-Approaches-4-Invertebrate-Biodiversity-Monitoring/blob/main/notebooks/notebook2.md) — Bioinformatics Pipeline

Core processing steps from raw reads to ASV table:

* Quality control and filtering
* Denoising and ASV inference
* Construction of the ASV abundance table
* Taxonomic assignment and filtering

### 📊 [Part 3](https://github.com/MontagnaLab/Innovative-Approaches-4-Invertebrate-Biodiversity-Monitoring/blob/main/notebooks/notebook3.md) — Ecological Analyses

Example downstream analyses, including:

* Alpha and beta diversity
* Community composition
* Basic ecological comparisons among samples

---

## Notes

This pipeline is intended for educational purposes but can be adapted for real research projects with appropriate parameter tuning and validation.
